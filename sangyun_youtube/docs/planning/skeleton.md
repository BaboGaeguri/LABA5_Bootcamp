# 영상 기획 스켈레톤

영상을 기획할 때 반드시 채워야 할 핵심 스펙과 구성 요소.

---

## 1. 영상 스펙

| 항목 | 설명 |
|------|------|
| **길이** | 몇 분짜리인지. 편집량과 정보 밀도가 여기서 결정됨. |
| **구조** | 섹션을 몇 개로 나누고, 각 섹션에 몇 초/분 배분하는지. |
| **훅** | 첫 몇 초에 뭘 보여주는지. 이게 약하면 나머지가 의미 없음. |
| **썸네일 + 제목** | 클릭을 만드는 요소. 영상 외부에서 싸우는 구간. |
| **엔딩** | 다음 편 연결 or 구독 유도를 어떻게 처리하는지. |

---

## 2. 기획 핵심 요소

| 항목 | 설명 |
|------|------|
| **중심 갈등/긴장** | "이게 될까?" 라는 질문을 영상 내내 유지. 시청자가 끝까지 볼 이유. |
| **감정 곡선** | 기대 → 시도 → 실패 → 반전 → 성공(or 깨달음). 평탄하면 지루함. |
| **정보 밀도 조절** | 설명이 너무 많으면 강의가 됨. 보여주고 나레이션으로 보완. |
| **엔딩의 여운** | 마지막 한 마디 or 장면이 다음 영상 클릭을 만듦. |

---

## 3. 에피소드 기획 시 체크리스트

- [ ] 길이 확정
- [ ] 섹션 구조 + 타임코드 배분
- [ ] 훅 장면 확정
- [ ] 썸네일 컨셉 + 제목 유형 확정
- [ ] 중심 갈등 설정
- [ ] 감정 곡선 흐름 확인
- [ ] 엔딩 + 다음 편 연결 확정

---

## 4. 시간 흐름 구성 방식

영상이 시간축을 어떻게 다루는지 미리 결정.

| 방식 | 설명 | 사용 상황 |
|------|------|---------|
| **Chronological (시간순)** | 실제 일어난 순서 그대로 보여줌 | 일상 브이로그, Day 단위 기록 |
| **훅 선행 (역삽입)** | 결말/하이라이트를 맨 앞에 배치 후 과정으로 돌아옴 | 훅이 강할 때. EP4처럼 성공 장면이 인상적일 때 |
| **병렬** | 두 흐름을 번갈아 보여줌 | 대조 구성 (챌린지 하는 나 vs 안 하는 친구) |
| **몽타주** | 시간 압축. 타임랩스·빠른 컷으로 긴 시간을 짧게 | 반복 루틴, 지루한 대기 구간 |

> EP4 적용: **훅 선행** (Gazebo 성공 장면 먼저) → **Chronological** (과정 전개) → **몽타주** (실패 반복 구간)

---

## 5. 타임라인 트랙 구성

영상 편집 툴의 타임라인을 어떻게 레이어로 나눌지 사전 설계.

| 트랙 | 용도 | 규칙 |
|------|------|------|
| **Video 1** | 메인 영상 (현장 촬영, 화면녹화) | 기본 화면. 항상 채워져 있어야 함. |
| **Video 2** | 오버레이 (자막 그래픽, 손그림, 챕터 타이틀) | 메인 위에 얹는 레이어. |
| **Audio 1 — BGM** | 배경음악. 영상 전체 깔리는 분위기 음악 | 볼륨 낮게 유지. 장면 전환 시 페이드인/아웃. |
| **Audio 2 — SFX** | 효과음. 임팩트 컷, 성공 순간, 실패 컷 등 포인트에만 사용 | 짧고 강하게. BGM과 겹치지 않게 볼륨 조절. |
| **Audio 3 — VO** | 나레이션 (목소리). 독백, 설명 멘트 | 가장 볼륨 높게. BGM은 VO 구간에서 자동 덕킹. |

> **덕킹(Ducking):** VO가 나올 때 BGM 볼륨을 자동으로 낮추는 기법. 나레이션이 묻히지 않게.

## 6. 클립 길이 원칙

**핵심 법칙: 한 클립 = 한 정보 단위.**
정보가 전달되는 순간 컷. 더 보여줄 게 없으면 1초라도 끊는다.
클립이 루즈한 이유는 길어서가 아니라 그 안에 새로운 정보가 없어서다.

### 장면 성격별 적정 클립 길이

| 장면 성격 | 적정 길이 | 이유 |
|---------|---------|------|
| **훅 / 임팩트 컷** | 1~3초 | 강하게 치고 바로 넘어가야 긴장감 유지 |
| **행동 / 과정 샷** | 3~8초 | 뭘 하는지 보이는 순간까지만. 그 이상은 루즈 |
| **개념 설명** | 5~15초 | 시청자가 이해할 시간 필요. 단 정보 없는 구간 제거 |
| **감정 순간 (성공·반응)** | 5~10초 | 여운을 줘야 하므로 예외적으로 길게 허용 |
| **실패 몽타주** | 1~3초 | 빠르게 쌓아야 긴장감·도파민 효과 |
| **타임랩스** | 2~3초 | 길면 의미 없어짐 |

### 나레이션 밀도 (섹션 성격별)

| 섹션 성격 | 밀도 | 기준 |
|---------|------|------|
| **훅** | 높음 | 3초당 나레이션 2개 |
| **개념 설명** | 낮음 | 5~7초당 1개. 이해할 시간 확보. |
| **실패 몽타주** | 높음 | 짧은 나레이션 + 빠른 컷 |
| **성공 시연** | 없음 or 1개 | 현장음이 주인공. 나레이션이 감동 방해 |
| **마무리** | 낮음 | 여운 위해 천천히. 문장 사이 공백 허용 |

> 쇼츠 기준: 3초에 나레이션 2개가 한계. 짧은 드립이면 예외적으로 +1~2개까지 허용.
