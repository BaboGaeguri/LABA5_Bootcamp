## 2. 기획 핵심

| 항목 | 외전C 적용 |
|------|---------|
| **중심 갈등** | "이 여행의 목표였던 모터, 화창베이에서 구할 수 있을까?" |
| **감정 곡선** | 미션 제시(모터 스펙 확인) → 일정이 꼬이며 생긴 여유(원래 Day3 예정이던 화창베이를 Day2에 당겨감) → 화창베이 스케일에 압도 → 발로 뛰며 사냥하지만 결국 실패 → 시간 걸려도 결국 알리익스프레스로 결정 → 탈진해서 녹초로 귀가 → 예상 못한 삼겹살 회식의 감동 → 2차 회식·주사위게임으로 텐션 최고조 → Day3 홍콩 이야기로 여운과 다음 기대감 |
| **정보 밀도** | 모터 스펙·화창베이 스케일은 화면+자막+짧은 나레이션. 탐험 실패 과정은 몽타주로 압축. 회식·게임 파트는 현장음 위주로 나레이션 최소화(사람 감정이 주인공). |
| **엔딩 여운** | "그날 우리가 못 찾은 건 모터였지만, 대신 찾은 게 있었다." — 실패(일)와 감동(사람)을 대비시켜 마무리. Day3 홍콩 이야기로 다음 여정 기대감까지 얹기. |

> **본편 EP7과의 연결 재조정 필요:** 회의자료(04_상윤_회의자료.md)는 "EP7의 불만에 외전C가 완벽한 해답"이라는 전제였으나, 실제로는 화창베이 소싱이 실패하고 알리익스프레스로 돌아왔음. EP7 떡밥 회수를 "여기서 다 구해왔다"가 아니라 "그래도 여기 와봐야 이유를 알 수 있었다"는 톤으로 수정 필요 — todo 참고.

> **외전B와의 중복 확인 필요:** 외전B가 "심천1, 도시+식도락"을 다루는데 외전C에 삼겹살 회식 비중이 커짐 — 식도락 소재가 겹치지 않는지 교통정리 필요 (todo 참고).

## 3. 고재영 스타일 적용

| 특징 | 외전C 적용 방식 |
|------|-------------|
| **모든 건 독백** | 팀원 대화 없이 나레이션으로만 전달. 시청자에게 직접 말 거는 구조. |
| **대주제 계속 강조** | "이번 여행의 목표는 모터였다"를 탐험 파트 내내 환기하다가, 후반부에서 "그런데 진짜 얻은 건 따로 있었다"로 전환. |
| **소주제 여러 개** | 일정 변경 → 화창베이 스케일 → 탐험(실패) → 귀가 → 삼겹살 회식 → 2차 회식·주사위게임 → Day3 예고 순서로 흐름. |
| **나레이션 + 영상** | 상가 골목 이동·흥정·회식·게임 장면 위에 나레이션 덧붙임. |
| **한 번 보여준 건 스킵** | 상가 골목 헤매는 장면은 처음만 자세히, 이후 반복 이동은 몽타주로 압축. |
| **대조 구성** | 화창베이 탐험의 실패(일) vs 삼겹살 회식의 감동(사람) 대비. 계획했던 Day3 화창베이 vs 실제로 당겨진 Day2 일정 대비. |
| **느낀 점** | 회식 자리에서 팀원들의 솔직한 반응(허탈함 → 웃음 → 다음 목적지 기대) 인터뷰로 처리. |
| **하루 마무리 멘트** | Day2 전체(탐험+회식+게임)를 정리하는 한마디 + Day3(홍콩) 예고로 하루를 닫음. 레퍼런스 영상의 "일기장 쓰듯 하루 소감" 구조를 그대로 적용. |
| **문장 사이사이 컷편집** | 나레이션 말하는 장면은 모든 문장 사이 컷. 여백 없음. |
