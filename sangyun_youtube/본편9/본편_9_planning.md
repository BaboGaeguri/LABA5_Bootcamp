## 3. 고재영 스타일 적용

| 특징 | 본편9 적용 방식 |
|------|-------------|
| **모든 건 독백** | 팀원 대화 없이 나레이션으로만 전달. 시청자에게 직접 말 거는 구조. |
| **대주제 계속 강조** | "혼자선 못 만든다"를 섹션마다 한 번씩 환기. |
| **소주제 여러 개** | 외형 제작(나연님 만남 → 디자인센터 미팅 → 소재/제작) → 트레이드오프 → 심천 협업자 → 의문의 협업자 순서로 흐름. |
| **나레이션 + 영상** | 미팅 현장, 카카오톡 대화, 3D 프린터 돌아가는 장면 위에 나레이션 덧붙임. |
| **한 번 보여준 건 스킵** | 디자인 버전 변천은 처음만 자세히, 이후 버전 변경은 타임랩스로 압축. |
| **대조 구성** | 골격 상태 하이리온 vs 외형 완성 하이리온 / 기획 디자인 vs 실제 제작 결과. |
| **느낀 점** | 완성 직후 팀원 반응 + "이만큼 사람이 필요했다" 복선으로 처리. |
| **문장 사이사이 컷편집** | 나레이션 말하는 장면은 모든 문장 사이 컷. 여백 없음. |

### 핵심 섹션 6개 (+ 인트로 + 마무리)

| # | 핵심 섹션 | 역할 | 고민 포인트 |
|---|---------|------|-----------|
| 1 | **훅** | 완성 외형 반전으로 "어떻게 만들었지?" 유발. | 외형 전후 대조가 핵심 — 골격 vs 완성본 컷을 얼마나 임팩트 있게 보여줄지. |
| - | **시리즈 공통 인트로** | 채널 로고/타이틀. | 훅의 긴장감을 끊지 않게 5초 이내로. |
| 2 | **외형 소재 & 제작** | 어떻게 만들었는지 전체 흐름 공개. 머리(보닝→3D프린팅), 슈트(살과 근육), 티셔츠. | 제작 방법 변경(보닝→3D프린팅)의 이유를 드라마틱하게 풀 것. 실패한 시도도 보여줄 것. |
| 2a | ↳ **나연님과의 만남** | 협업 시작 선언 + 감정 입구. "우리끼리는 벅차다" 솔직한 고백. | 카카오톡 대화 공개 범위와 편집 방법. 나연님 소개를 어디까지 할지. |
| 2b | ↳ **디자인센터와의 미팅** | 외형 크기 결정 과정 + v6→최종 디자인 변천사. | 나연님에게 전달하기 위해 사전에 정해야 했던 스펙들을 시청자가 이해하게 만드는 순서 구성. |
| 3 | **디자인 vs 기능 트레이드오프** | 갈등 구간. 예뻐야 하는데 기능이 막는다. 머리 크기/무게, 카메라 위치, 열관리, 프린터 시간싸움. | 각 트레이드오프를 짧게 치고 빠져야 함. 하나에 너무 오래 머물면 지루해짐. |
| 4 | **심천의 협업자들** | 심천에서 만난 협업자들 소개. 부품 소싱 및 현지 네트워크 이야기. | 외전C(화창베이)와 연결되는 맥락 짧게 리마인드. 심천 현장감 살리기. |
| 5 | **의문의 협업자들** | 뜻밖의 인물들 소개로 도파민 유발. 미국인, 모터구, CITE 대학생, 태영형, 아마도 개발자. | 각 인물이 어떻게 연결됐는지 짧게 설명. 너무 길면 산만해짐 — 인물당 15~20초 이내. |
| - | **마무리** | 소감 + "협업 없이는 없었다" + 완성 외형 하이리온 전신 샷 + 다음 편 예고. | 여운 남기고 끊기. 엔드스크린까지 긴장감 유지. |
