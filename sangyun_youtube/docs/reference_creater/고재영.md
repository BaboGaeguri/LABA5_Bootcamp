# 레퍼런스 크리에이터: 고재영

EP4는 고재영 스타일을 베이스로 기획한다.

---

## 고재영 스타일 특징

| # | 특징 | 설명 |
|---|------|------|
| 1 | **모든 건 독백** | 대화 없이 혼자 말하는 구조. 시청자에게 직접 말 거는 느낌. |
| 2 | **설치해놓고 이동하는 거 찍기** | 카메라를 고정 설치한 채 자연스럽게 행동. 연출 없이 일상처럼. |
| 3 | **느낀 점 이야기** | 사건/행동 후 감정·생각을 솔직하게 말함. |
| 4 | **큰 주제 안에 소주제 여러 개** | 하나의 대주제 아래 작은 에피소드들을 연속으로 보여줌. (ex. 동사무소 방문, 밥 먹기 등) |
| 5 | **나레이션 + 영상 처리** | 뭔가 행동하는 장면은 나레이션(말하기)을 덧붙여 처리. 설명과 영상이 동시에. |
| 6 | **타임랩스 2~3초** | 시간 경과는 짧은 타임랩스로 처리. 길게 끌지 않음. |
| 7 | **하루 마무리 멘트** | 에피소드 끝에 그날을 정리하는 한마디. |
| 8 | **대주제를 계속 강조** | 영상 전체에서 핵심 주제를 반복적으로 상기시킴. |
| 9 | **한 번 보여준 내용은 스킵** | 이미 설명한 개념·장면은 재설명 없이 그냥 넘어감. 시청자를 신뢰. |
| 10 | **챌린지 하는 것 vs 안 하는 것 대조** | 본인의 행동과 주변(친구 등)의 행동을 대비시켜 보여줌. |
| 11 | **나레이션·말하기는 문장 사이사이 컷편집** | 말하는 장면은 문장과 문장 사이를 칼같이 컷. 여백·망설임 없음. |

## 핵심 편집 원칙 (EP4 적용)

**반복되는 하루 단위는 처음에만 자세히, 이후엔 컷편집으로 빠르게 스킵.**

- **Day 1:** 루틴 전체를 처음 한 번 자세히 보여준다.
- **Day 2 이후:** 같은 루틴은 컷편집으로 1~2초 안에 처리. 핵심 변화나 감정만 남긴다.
- 시청자가 패턴을 이해한 순간부터는 설명 없이 진행. 지루함 없이 밀도 유지.
- 새로운 소주제(장소 이동, 친구 만남 등)가 생길 때만 다시 자세히 보여준다.
